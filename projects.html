📈 Parametric Survival Analysis — Rats Dataset

Full survival analysis pipeline applied to the rats dataset in R, modelling time-to-tumour onset following carcinogenic treatment. Covers Kaplan–Meier estimation, Cox regression, Weibull AFT modelling, parametric model selection across 7 distributions, life function estimation, and a shared gamma frailty model to account for litter clustering.

R Survival Analysis Weibull Cox Regression Frailty Model flexsurv survminer

Key result: Weibull model best fit (AIC = 578.21), treated rats have 2× the tumour hazard of controls (HR = 2.21, p = 0.011)
